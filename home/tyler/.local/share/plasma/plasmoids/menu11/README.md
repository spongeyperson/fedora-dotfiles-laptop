<h2 align="center">Installing Manually</h2>

### Using git
```bash
git clone https://github.com/prateekmedia/Menu11.git ~/.local/share/plasma/plasmoids/menu11;
kquitapp5 plasmashell || killall plasmashell && kstart5 plasmashell;
```
